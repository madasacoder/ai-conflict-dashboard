
# AI Conflict Dashboard

## Project Structure
- Frontend: React + TypeScript
- Backend: FastAPI (Python 3.11+)
- Docs: Strategic and technical guidelines

## Setup
### Frontend
```bash
cd frontend
npm install
npm run start
```

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
